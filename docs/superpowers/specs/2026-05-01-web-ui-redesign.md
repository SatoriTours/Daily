# Daily Satori Web 管理后台 UI 重设计

## 1. 概述

将现有的管理后台 Web 界面从"工具化风格"改造为**舒服自然、经典优雅**的个人知识库风格。

### 核心风格方向
- **E-Ink / Paper**：纸质书阅读器般的自然感，柔和、哑光、不刺眼
- **Bento Grids**：仪表盘模块化卡片布局
- **Soft UI Evolution**：兼顾柔和感与现代可访问性

## 3. 图标

**全部替换为 SVG 图标，不再使用 Unicode/Emoji 字符。**

使用 **Lucide** 图标集，内联 SVG 方式嵌入。具体图标映射：

| 位置 | 当前 | 替换为 SVG |
|------|------|-----------|
| 品牌图标 | ◆ | `book-open` (书) |
| 仪表盘 | ◫ | `layout-dashboard` |
| 文章 | ☰ | `file-text` |
| 日记 | 📓 | `book-marked` |
| 书籍 | 📚 | `library` |
| 退出 | ↩ | `log-out` |
| 浅色/深色 | ☀/☾ | `sun` / `moon` |
| 展开/折叠 | ▶/◀ | `chevron-right` / `chevron-left` |
| 搜索 | - | `search` (添加在搜索框前) |
| 统计图标 | Emoji | `globe`(文章), `book-marked`(日记), `library`(书籍), `star`(收藏) |

## 4. 布局架构

### 4.1 侧边栏

- 宽度：220px（折叠 60px）
- 背景色融入页面：浅色 `#F0ECE0`，深色 `#1A1715`
- 不再使用固定的深色侧边栏
- 导航项增加圆角 8px，hover 有柔和背景变化
- active 状态：主色文字 + 半透明主色背景，非实色填充
- 底部增加用户信息和主题切换，整体更轻盈

### 4.2 顶栏

- 高度 56px
- 背景与内容区一致：`#FFFDF7`
- 底部有柔和分割线
- 标题使用 Lora 字体
- 搜索框在右侧，前置 search 图标，圆角 8px

### 4.3 内容区

- padding: 24px（移动端 16px）
- 背景色即为页面整体背景

## 5. 页面设计

### 5.1 登录页

- 全屏居中，背景使用微妙的纸张纹理/噪点
- 大标题 "Daily Satori" 使用 Lora 字体，字号 28px
- 副标题 "个人知识管理" 使用 Inter，次要色
- 卡片居中，padding: 40px，阴影 sm
- 输入框圆角 8px，focus 时边框变主色
- 登录按钮全宽，主色填充

### 5.2 仪表盘

#### 统计卡片（Bento 风格）
- 4 个卡片，使用 CSS Grid 自适应布局
- 每个卡片：温和背景 + 左上角 SVG 图标 + 数值 + 标签
- 图标使用柔和底色圆形背景（16px 圆角方形）
- 数值使用 Inter 600 字重，24px 大号
- 卡片 hover 有微妙阴影提升

#### 内容趋势图
- 使用 Chart.js 的 Line Chart
- 柔和配色，低饱和度线条
- 去掉网格线的生硬感，使用虚线或更浅色
- 图例在底部

#### 最近活动
- 简洁时间线风格
- 左侧圆点（类型色）+ 标题 + 右侧时间
- 列表项之间有细微分割

### 5.3 文章管理

- 卡片网格布局（Bento 风格）
- 封面图区域：圆角 12px（上方），无封面时显示占位（柔和灰底 + 书籍图标）
- 标题使用 Lora 字体
- 摘要文字 2 行截断
- 卡片 hover：阴影 + 轻微上浮 (translateY -2px)
- 分页器居中，圆角按钮

### 5.4 日记管理

- 列表式卡片，每个日记一张卡片
- 卡片左侧有日期竖线装饰
- 内容预览（Markdown 渲染后截取）
- 图片缩略图圆角 8px，hover 放大效果
- 标签使用暖色系柔和底板

### 5.5 书籍管理

- 左右分栏布局不变，但视觉优化
- 左侧书架列表：圆角列表项，选中项左侧有主色竖条标识
- 右侧详情区：大标题 Lora + 作者信息
- 读书感悟使用折叠面板 (details/summary)，展开时有微动画

## 6. 组件细节

### 6.1 按钮

- 圆角 8px
- padding: 8px 16px (默认), 4px 10px (sm)
- 主按钮：主色填充 + 白色文字
- 次按钮：边框 + 透明背景
- 危险按钮：危险色填充
- hover 时背景色加深 10%
- disabled 状态：opacity 0.5

### 6.2 输入框

- 圆角 8px
- 边框 1px solid，focus 时边框变主色
- 背景色略深于卡片背景
- placeholder 颜色用次要色

### 6.3 模态框

- 圆角 16px
- 头部底部有边框分割
- 背景遮罩：rgba(0,0,0,0.4)
- 关闭按钮：圆形 + hover 背景
- 进入动画：fadeIn + slideUp

### 6.4 Toast

- 圆角 8px
- 轻柔阴影
- 右上角出现
- 成功：成功色背景 + 白字
- 错误：危险色背景 + 白字

### 6.5 标签 (Tag)

- 圆角 6px
- 柔和背景色，不使用纯灰
- 字体 11px

## 7. 响应式

| 断点 | 行为 |
|------|------|
| ≥1024px | 完整布局，侧边栏 220px |
| 769-1023px | 文章卡片 3 列 → 2 列，侧边栏保持 |
| ≤768px | 侧边栏自动折叠 (60px)，文章卡片 1 列 |
| ≤480px | 全宽，统计卡片 1 列，模态框 95% 宽度 |

## 8. 深色模式

- 使用 `data-theme` 属性切换
- 系统偏好自动检测 (`prefers-color-scheme`)
- localStorage 记住用户选择
- 所有颜色均有浅/深两套变量
- 深色下阴影减弱但不变暗（用透明度替代）
