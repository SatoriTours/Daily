# v5.0.17 更新日志

- 新增远程文章详情双 Tab 阅读体验，可在 AI 摘要与原文之间切换。
- 优化每日新闻汇总质量，生成前会过滤低信息量内容、合并重复来源并保留来源多样性。
- 优化每日新闻汇总提示词结构，改为今日要点、重要变化、值得关注的简报格式。
- 优化新闻引用校验，避免无效引用、Markdown 链接标签或无引用内容被当作可靠来源。
- 优化部分新闻源失败时的提示，保留可用来源继续生成汇总并显示更清晰的状态。
- 优化项目结构，清理 Flutter 遗留工程、测试脚本和发布流程，统一为 KMP/Android 构建链路。
- 修复远程新闻详情组件职责过重的问题，拆分摘要详情、文章详情和卡片组件。
- 修复每日汇总中重复 URL、重复标题来源和桥接重复项未完全合并的问题。
- 修复每日汇总在所有远程来源失败时误显示为空内容的问题。
