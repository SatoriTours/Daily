# v5.0.16 更新日志

- 新增多远程新闻源配置能力，新闻汇总可统一轮询多个远程来源并生成本地 AI 总结。
- 新增远程新闻完整接口 URL 配置方式，兼容旧根地址自动迁移到 `top_articles_today` 接口。
- 优化远程新闻设置页，改为列表页与编辑页两级结构，提升配置和维护体验。
- 优化远程新闻详情阅读页，增加标题卡、来源标签、摘要、关键观点、正文与原文链接分区。
- 优化新闻汇总重新生成反馈，空结果或失败时显示明确提示，不再表现为无响应。
- 修复远程新闻 Token 测试误判成功的问题，连接测试会校验接口返回并显示获取到的文章数。
- 修复远程文章引用详情在完整 URL 配置下拼接错误的问题。
