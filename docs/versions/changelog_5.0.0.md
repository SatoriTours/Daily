# v5.0.0 更新日志

## 新增

- 新增 Android KMP/Jetpack Compose 版本发布流程，支持通过 GitHub Actions 构建并发布 APK。
- 新增 AI 助手记忆系统，支持从文章、日记和读书内容中检索引用来源。
- 新增 AI 助手引用详情，文章可跳转详情页，日记和读书内容可在底部弹窗中查看。
- 新增 Web 服务与桌面端管理界面，支持本地资源服务、认证、统计和内容查看。
- 新增文章、日记、读书、AI 配置、备份恢复等核心模块的 Compose 页面和 Koin ViewModel 架构。

## 优化

- 优化文章接收、解析、图片提取、AI 总结和失败恢复流程，提升导入和刷新可靠性。
- 优化 AI 配置流程，支持多服务商模型预设、默认模型策略和连接测试。
- 优化文章、日记、读书和设置页面的视觉样式、交互体验和 Markdown 渲染。
- 优化 Web 服务启动、端口检测、静态资源打包和 API 鉴权逻辑。
- 优化数据迁移、备份加密、旧版图片迁移和本地图片访问能力。

## 修复

- 修复 AI 助手总结失败时暴露内部错误的问题，并增加静默重试。
- 修复导入 Flutter 旧版数据时无时区 ISO 时间解析失败，导致文章、日记和书籍创建时间变成导入时间的问题。
- 修复文章详情、日记图片显示、本地图片路由、Web 登录和服务开关状态等问题。
