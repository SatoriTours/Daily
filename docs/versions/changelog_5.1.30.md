# v5.1.30

- 新增统一异步任务框架，支持任务注册、进度查询、失败重试、应用回到前台后继续重试，并接入文章处理、新闻同步、X 收藏同步和书籍观点生成等流程。
- 新增设置页任务中心入口，默认展示未完成任务，并支持按任务类型和状态过滤查看执行情况。
- 新增文章详情页右上角复制网页链接功能，并为 X 文章提供通过 X API 刷新内容的入口。
- 优化 X 收藏同步对长文、引用和转发内容的解析，支持从引用子节点和 `x.com/i/article` 链接补全文章正文。
- 优化文章刷新和 AI 整理语气，减少“这是一篇文章”这类第三视角开场，让内容更像直接阅读正文。
- 修复删除已导入文章后无法重新同步的问题，并让异步同步任务真实展示处理进度。
