# Daily Satori（每日顿悟）

Daily Satori是一款专注于个人知识管理和学习的辅助工具，帮助您高效收藏、整理和回顾有价值的网络内容。应用采用本地优先设计，确保快速响应和隐私保护。

## ✨ 核心功能

### 📑 文章收藏与管理
- **智能收藏**：一键保存网页内容，自动提取标题、正文和关键图片
- **分类整理**：通过标签系统组织您的收藏内容，快速建立个人知识库
- **全文搜索**：强大的搜索引擎，支持标题、内容、标签多维度检索
- **收藏夹**：将重要文章标记为收藏，创建自定义阅读列表
- **批量操作**：支持多选文章进行批量标记、移动和删除操作 --还未实现

### 🤖 AI辅助功能
- **内容摘要**：自动生成文章摘要，快速了解核心内容
- **智能解读**：AI分析文章重点，提供关键洞见和个性化学习建议
- **内容转换**：将HTML内容转换为Markdown格式，便于阅读和编辑
- **智能标签**：自动为文章生成标签建议，减少手动分类工作

### 📱 离线阅读体验
- **内容本地化**：所有文章内容保存在本地，无需联网也能阅读
- **图片缓存**：自动缓存文章中的图片，确保离线时完整体验
- **纯本地存储**：所有数据存储在本地，无需担心服务中断
- **阅读模式**：支持日间/夜间模式切换
- **稍后阅读**：一键保存网页至离线阅读列表，随时随地享受优质内容

### 🔄 跨平台使用
- **Web服务器**：内置Web服务器功能，可通过浏览器在电脑上访问收藏内容
- **备份恢复**：支持数据备份和恢复，确保数据安全
- **自动更新**：应用内检测新版本，一键更新
- **多设备同步**：通过自托管方式实现多设备数据同步
- **桌面客户端**：计划推出Windows/Mac客户端，提供更完整的桌面体验 --还未实现

### 📝 创作与笔记
- **Markdown编辑器**：支持Markdown语法的轻量级编辑器
- **日记系统**：内置日记功能，记录每日感悟和学习心得
- **知识整理**：将收藏内容与个人笔记关联，构建完整知识体系  --还未实现
- **思维导图**：可视化展示知识关联，帮助理解复杂概念        --还未实现

### 📚 读书学习
- **书籍管理**：整理您的阅读书籍，记录阅读进度和心得        --还未实现
- **观点提取**：AI自动从书籍中提取关键观点和精华内容
- **观点卡片**：将书中重要观点制作成卡片形式，方便复习和分享
- **分类整理**：按作者、分类组织您的书籍库，构建个人知识体系 --还未实现
- **阅读笔记**：记录阅读过程中的思考和感悟，加深理解        --还未实现
- **书籍简介**：自动生成书籍简介，帮助快速了解书籍内容
- **封面展示**：美观的书籍封面展示，让您的书架更加直观      --还未实现

## ⚙️ 高级设置

### OpenAI集成
如需启用AI摘要和解读功能，可在设置中配置OpenAI API密钥。应用支持自定义API端点，兼容多种AI服务提供商。

### Web服务器
在设置中配置Web服务器地址和密码后，可通过浏览器在电脑上访问和管理收藏内容。支持HTTPS安全连接和访问权限控制。

### 插件系统
支持通过[插件系统](https://github.com/SatoriTours/plugin)扩展应用功能，自定义您的使用体验。目前提供内容处理、UI扩展和数据导出等多种插件类型。

## 💻 技术架构

Daily Satori采用现代Flutter架构开发，主要技术栈包括：

- **前端框架**：Flutter
- **状态管理**：Riverpod
- **本地存储**：SQLite + Isar数据库
- **网络处理**：Dio + HTML解析
- **AI集成**：OpenAI API
- **Web服务**：内置轻量级HTTP服务器

应用遵循整洁架构设计原则，确保代码可维护性和可扩展性。

## ❓ 常见问题

### Q: 应用数据保存在哪里？
A: 所有数据默认保存在设备本地存储中，可在设置中查看和修改数据存储路径。

### Q: 如何备份我的数据？
A: 在设置中选择"备份与恢复"，可将数据导出为加密备份文件，支持手动备份和自动定时备份。

### Q: 无法连接到Web服务器怎么办？
A: 请确保移动设备和电脑在同一网络环境下，并检查防火墙设置是否允许应用的网络访问。

### Q: 如何安装和使用插件？
A: 在设置中选择"插件管理"，可浏览和安装官方认证的插件，或导入自定义插件包。

## 📚 隐私说明
Daily Satori采用本地优先设计，所有数据存储在您的设备上。AI功能调用仅在您主动开启并配置的情况下进行。应用不会收集任何个人使用数据或上传内容到外部服务器。
